Introduction to Internet

Internet is a means of connecting a computer to any other computer anywhere in the world via dedicated routers and servers. When two computers are connected over the Internet, they can send and receive all kinds of information such as text, graphics, voice, video, and computer programs.

Types of Networks

Intranet is a private network accessible only to an organization's staff.

Extranet is a controlled private network allowing customers, partners, vendors, suppliers and other businesses to gain information, typically about a specific company or educational institution, and do so without granting access to the organization's entire network. An extranet is often a private part of a website. It is restricted to select users through user IDs, passwords and other authentication mechanisms on a login page.

Computer Networks

A computer network is a collection of various computing devices in a way that they communicate with each other. A device is also called a node, which could be a personal computer (PC), a server, an Internet Protocol (IP) phone, a smart phone connecting to the network via a Wi-Fi router, a printer etc.

The devices are connected to each other using a variety of media. In a network, more than one type of media can be used wired (also called guided) or unwired (also called unguided).

Network Cables

  • Twisted pair - used to make local area network (LAN) cables that connect nodes to the network. The common term is network cable, Ethernet cable or RJ45 cable
  • Coaxial cable - old technology and is hardly used. It is same as a television aerial cable. The transmission speeds were low and the termination complex to make
  • Fiber cables - used for backbone connectivity (e.g. between buildings or cities). They are also used in the patch panel connectivity in the server room and inside the systems unit to connect the hard disc drive to the motherboard

Electronic Mail (Email)

Electronic mail (email, e-mail) is a system for sending messages from one individual to another via telecommunications links between computers or terminals using dedicated software. It also means the document sent by email software which may include embedded images or attached files.

Popular Email Software

  • Microsoft Outlook (or Outlook Express) by Microsoft Inc.
  • Mozilla Thunderbird by Mozilla Foundation
  • Gnome Evolution, a Linux/Gnome based email and calendar client (there is now a windows version of Evolution)
  • Kontact, a Linux/KDE based email and calendar client

Email Protocols

Standards that govern how email is sent, received, and stored:

Protocol Purpose Function
POP3 (Post Office Protocol version 3) Receive emails from a remote server to a local email client Allows you to download email messages on your local computer and read them even when you are offline
IMAP (Internet Message Access Protocol) Access email on a remote web server from a local client Used for retrieving emails, keeps messages on the server
SMTP (Simple Mail Transfer Protocol) Send emails from a client to a server or between mail servers Works with outgoing mail only; cannot retrieve emails

Web Browsers

A web browser is software that is used in retrieving, presenting, and traversing information resources on the Internet. There are different types of browsers but all use Uniform Resource Locator (URL) also called Uniform Resource Identifier (URI), which is a unique address to a web page.

Leading Internet Browsers

  1. Google Chrome
  2. Mozilla Firefox
  3. Apple Safari
  4. Microsoft Internet Explorer
  5. Microsoft Edge
  6. Opera
  7. Maxthon

Google Chrome has been gaining popularity over time, while Microsoft's Internet Explorer has been losing popularity.

Uniform Resource Locator/Identifier (URL/URI)

A URI is a string of characters used to identify a resource on the Internet. It is the global address of documents and other resources on the World Wide Web.

A URL is one type of Uniform Resource Identifier (URI); the generic term for all types of names and addresses that refer to objects on the World Wide Web.

A Uniform Resource Locator (URL), commonly informally termed a web address is a reference to a web resource that specifies its location on a computer network and a mechanism for retrieving it.

The URL contains the name of the protocol to be used to access the resource and a resource name. The first part of a URL identifies what protocol to use. The second part identifies the IP address or domain name where the resource is located.

Configuring Web Browsers

Browser setting enables a user to define how the application should interact with the real world. Attackers use the Internet to compromise computers through vulnerable web browsers.

An insecure web browser can lead to spyware being installed on your computer without your knowledge, attackers taking control of your computer, stealing your information, or even using your computer to attack other computers.

Of all the software out there, browsers are the most exposed and information rich. If a malicious hacker manages to break in, he can find out just about anything about you: address, credit card data, emails, passwords, browsing history, bookmarks etc.

Configuring Google Chrome

To configure the setting for Google Chrome:

  1. Left click on the three vertical buttons
  2. The settings window loads
  3. Sign in to Google Chrome - enables you to log on to your Google mail account
  4. Default Browser - enables automatic opening of pages

Configuring Cookies

"Cookies" are small pieces of data that a website stores on your computer or device through your web browser.

Function: Cookies help websites remember your preferences, login information, and past actions on the site, making your future visits smoother.

How they work: When you visit a website, it sends a cookie to your browser which is then stored on your device.

Class Exercise: Configuring Cookies

  1. Open Google Chrome home page (www.google.co.ke)
  2. Select Settings → Show Advanced Settings → Privacy → Content Settings
  3. To allow first-party and third-party cookies, choose "Allow local data to be set"
  4. To block all cookies, choose "Block sites from setting any data"
  5. To allow only first-party cookies and block all third-party cookies, check the box next to "Block third-party cookies and site data"

Configuring Pop-Ups

Pop-up ads or pop-ups are often forms of online advertising on the World Wide Web intended to attract web traffic or capture email addresses.

By default, Google Chrome blocks pop-ups from automatically showing up on your screen. When a pop-up is blocked, the address bar will be marked "Pop-ups blocked". You can also decide to allow pop-ups.

If you still get pop-ups after disabling them, then you could have malware.

Class Exercise: Configuring Pop-Ups

  1. On your computer, open Chrome and select Settings
  2. At the bottom, click Show advanced settings
  3. Under "Privacy," click Content settings
  4. Under "Pop-ups," select an option: "Do not allow any site to show pop-ups (recommended)" or "Allow all sites to show pop-ups"

Browser Search

Searching for information is one of the most important uses of the Internet. There are many search engines that can be used to search for information on the Internet.

Search Engines

A search engine is a program that searches for and identifies items in a database that correspond to keywords or characters specified by the user, used especially for finding particular sites on the World Wide Web.

Search engines are used to find information on the Internet. They use automated software programs (known as bots, robots or spiders) that travel along the Web, following links from page to page, site to site.

Popular Search Engines

  • Google
  • Bing
  • Yahoo
  • Ask
  • DuckDuckGo

Searching Techniques

There are different ways of searching for information on the Internet. The following are some of the techniques:

  • Phrase searching - Use quotation marks to search for an exact phrase
  • Boolean searching - Use AND, OR, NOT to combine search terms
  • Truncation - Use an asterisk (*) to search for variations of a word
  • Wildcard searching - Use a question mark (?) to replace a single character
  • Field searching - Limit your search to specific fields such as title, author, etc.

Email Management

Creating an Email Account

To create an email account, you need to:

  1. Choose an email service provider (e.g., Gmail, Yahoo, Outlook)
  2. Go to the provider's website
  3. Click on the "Create account" or "Sign up" button
  4. Fill in the required information (name, username, password, etc.)
  5. Verify your account (usually through a phone number or another email address)
  6. Agree to the terms and conditions
  7. Click on the "Create account" button

Email Etiquette

Email etiquette refers to the principles of behavior that one should use when writing or answering email messages. It is also known as the code of conduct for email communication.

  • Use a clear and concise subject line
  • Use a proper salutation
  • Keep your email brief and to the point
  • Use proper grammar and spelling
  • Be professional and courteous
  • Use a signature block
  • Be careful with attachments
  • Respond to emails in a timely manner
  • Use "Reply All" sparingly
  • Avoid using all caps

Managing Email Messages

Email management involves organizing and maintaining your email messages. This includes:

  • Creating folders to organize your emails
  • Using filters to automatically sort incoming emails
  • Archiving old emails
  • Deleting unnecessary emails
  • Using labels or tags to categorize emails
  • Setting up rules for automatic email processing

Internet Security and Ethics

Information Security

Information security refers to the processes and methodologies which are designed and implemented to protect print, electronic, or any other form of confidential, private and sensitive information or data from unauthorized access, use, misuse, disclosure, destruction, modification, or disruption.

Common Internet Threats

  • Malware - malicious software designed to harm or exploit any programmable device, service or network
  • Phishing - attempting to acquire sensitive information by masquerading as a trustworthy entity
  • Spam - unsolicited messages sent in bulk by email
  • Spyware - software that enables a user to obtain covert information about another's computer activities
  • Adware - software that automatically displays or downloads advertising material
  • Ransomware - a type of malware that blocks access to the victim's data and threatens to publish or delete it unless a ransom is paid

Protecting Yourself Online

  • Use strong, unique passwords for each account
  • Enable two-factor authentication
  • Keep your software and operating system updated
  • Use a firewall and antivirus software
  • Be cautious when clicking on links or downloading attachments
  • Use a virtual private network (VPN) when on public Wi-Fi
  • Regularly back up your data
  • Be mindful of what you share on social media

Copyright and Intellectual Property

Copyright is a legal right that grants the creator of an original work exclusive rights to its use and distribution, usually for a limited time, with the intention of enabling the creator to receive compensation for their intellectual effort.

When using content from the Internet, it's important to:

  • Respect copyright laws
  • Give proper attribution when using someone else's work
  • Understand fair use guidelines
  • Use royalty-free or Creative Commons licensed content when possible

Q&A Session

Question: What is the difference between copyright and plagiarism?

Answer: Copyright is a legal concept that protects the rights of creators, while plagiarism is an ethical concept related to presenting someone else's work as your own without proper attribution.
