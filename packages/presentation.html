  • 1 What is Presentation Software?

    • Understanding Presentation Software

    Presentation software is an application used to prepare slide show based presentations with a graphical approach. A PowerPoint presentation helps complement oral presentations.

    Presentation software creates visual aids for presentations

    Key Functions of Presentation Software:

    • An editor that allows text to be inserted and formatted
    • A method for inserting and manipulating graphic images
    • A slide-show system to display the content
    Presentation Software Options: Microsoft PowerPoint, Google Slides, Apple Keynote, OpenOffice Impress, and Adobe Flash.
  • 2 PowerPoint Interface Overview

    • Navigating the PowerPoint Interface

    When you open PowerPoint, you'll see several key components that make up the interface:

    Stage/Canvas: Area used to design each PowerPoint slide by adding text, images, charts, graphs and tables
    Outline Pane: Contains the text from the slides. One can quickly locate a slide and by clicking on it from the outline pane, have it loaded on the stage for editing.
    Notes Pane: Used to provide additional details or notes that would not otherwise fit in the stage area, or that provide additional information for the slide.
    Slide Layout: Used to format the order of tables, charts, graphs and text on a slide.
    Ribbon: Collection of horizontal tabbed toolbars: Home, Insert, Design, Animations, Slideshow and Review.
    PowerPoint interface with key components labeled

    PowerPoint Views

    PowerPoint offers different views for creating and presenting your slides:

    Normal View

    Default view used to design presentations

    Slide Sorter View

    Changes slides to thumbnails for easy organization

    Notes Page

    Allows additional notes to be created or modified

    Slide Show View

    Full-screen presentation mode

  • 3 Creating Your First Presentation

    • Building a Basic PowerPoint Presentation

    Creating a presentation in PowerPoint is straightforward. Follow these steps to create your first presentation:

    Step 1: Start PowerPoint

    Start → Microsoft Office → Microsoft PowerPoint

    Step 2: Add Content to First Slide

    First Place Holder - Enter Your Name
    Second Place Holder - "My first PowerPoint Presentation"

    Step 3: Save Your Presentation

    File → Save As → Choose location
    File Name: My First Presentation - Your Name
    Save as Type: PowerPoint Presentation

    Step 4: Add New Slides

    Click on New Slide button or press Ctrl+M
    Select appropriate layout (Title and Content, Two Content, etc.)

    PowerPoint Templates

    A PowerPoint template is a pattern or blueprint of a slide or group of slides that you save as a .potx file. Templates can contain:

    Layouts

    Predefined arrangements of content placeholders

    Theme Colors

    Coordinated color schemes

    Theme Fonts

    Coordinated font combinations

    Background Styles

    Predefined background designs

    Quick Tip: To quickly create a professional-looking presentation, start with one of PowerPoint's built-in templates rather than a blank presentation.

  • 1 Inserting and Formatting Tables

    • Adding Tables to Your Presentation

    Tables are excellent for presenting structured data in a clear, organized manner. PowerPoint allows you to create and format tables directly within your presentation.

    Tables organize data in rows and columns for clear presentation

    Creating a Table

    Follow these steps to add a table to your presentation:

    Step 1: Insert Table

    Go to Insert tab → Table
    Select the number of columns and rows

    Step 2: Add Content

    Click in each cell and type your data

    Step 3: Format the Table

    Use Table Design tab to apply styles
    Use Layout tab to modify structure

    Table Modification Techniques

    Split Cells

    Divide a single cell into multiple cells

    Merge Cells

    Combine multiple cells into one

    Add Rows/Columns

    Insert new rows or columns as needed

    Apply Styles

    Use predefined table styles for professional look

    Table Style Tip: Use the Table Styles gallery in the Design tab to quickly apply professional formatting to your tables. Hover over different styles to see a live preview.
  • 2 Working with Excel in PowerPoint

    • Integrating Excel Data with PowerPoint

    PowerPoint allows you to embed Excel worksheets directly into your presentations, maintaining the functionality of Excel within PowerPoint.

    Excel worksheets can be embedded in PowerPoint slides

    Embedding Excel Worksheets

    To add an Excel worksheet to your presentation:

    Method 1: Insert Excel Spreadsheet

    Insert tab → Table → Excel Spreadsheet
    An embedded Excel worksheet will appear
    Enter data as you would in Excel

    Method 2: Copy from Excel

    In Excel, select and copy the data (Ctrl+C)
    In PowerPoint, paste using Paste Special
    Choose to keep source formatting or link to Excel

    Linking Excel Workbooks

    For data that updates frequently, you can create links between Excel and PowerPoint:

    Creating Linked Data

    Insert tab → Text grouping → Object
    Select "Create from File"
    Browse to your Excel file and check "Link" box
    Click OK to insert the linked object
    Important: When using linked Excel data, always close and restart PowerPoint to ensure links are updated properly. PowerPoint will prompt you to update links when reopening the presentation.

    Benefits of Linking vs. Embedding

    Linking

    Data updates when source Excel file changes

    Smaller file size

    Requires maintaining the source file

    Embedding

    Data is static (doesn't update)

    Larger file size

    Self-contained - no external files needed

Unit 3.1: Adding Graphics and Illustrations

  • 1 Images, Clip Art, and WordArt

    • Enhancing Presentations with Visual Elements

    Adding graphics to a presentation makes it easier to highlight important aspects and create visually appealing content that engages your audience.

    Graphics enhance the visual appeal of your presentations

    Inserting Pictures

    To add images from your computer to a presentation:

    Method 1: Insert Picture

    Insert tab → Images group → Pictures
    Browse to locate your image file
    Select and click Insert

    Method 2: Drag and Drop

    Open File Explorer and locate your image
    Drag the image file directly onto the slide

    Working with Clip Art

    Clip Art are pre-defined pictures and symbols that come with PowerPoint or can be downloaded from the Internet:

    Inserting Clip Art

    Insert tab → Images group → Online Pictures
    Search for relevant clip art
    Select and insert into your presentation

    Creating WordArt

    WordArt is a text-modifying feature that allows text to be presented with additional visual effects:

    Adding WordArt

    Insert tab → Text group → WordArt
    Select a WordArt style from the gallery
    Type your text in the WordArt box
    Use Format tab to customize appearance

    Image Formatting Options

    Crop

    Remove unwanted parts of an image

    Adjustments

    Modify brightness, contrast, and color

    Artistic Effects

    Apply special effects like blur, sketch, etc.

    Picture Styles

    Apply frames, shadows, and 3D effects

    Design Tip: Use high-quality images that are relevant to your content. Avoid stretching images disproportionately as this reduces quality. Use the crop tool to focus on the most important parts of an image.
  • 2 SmartArt and Charts

    • Visualizing Information with SmartArt and Charts

    SmartArt graphics and charts help you communicate information visually, making complex concepts easier to understand.

    SmartArt and charts help visualize relationships and data

    Working with SmartArt

    SmartArt graphics are visual representations of information that can quickly and effectively communicate your message:

    Creating SmartArt

    Insert tab → Illustrations group → SmartArt
    Choose a category and layout
    Enter text in the Text pane or directly in shapes

    SmartArt Categories

    List

    Show non-sequential information

    Process

    Show steps in a process or timeline

    Cycle

    Show a continual process

    Hierarchy

    Show hierarchical relationships

    Relationship

    Show connections between elements

    Matrix

    Show how parts relate to a whole

    Creating Charts

    Charts are effective for presenting numerical data in a visual format:

    Inserting a Chart

    Insert tab → Illustrations group → Chart
    Select chart type (Column, Line, Pie, etc.)
    Enter data in the Excel worksheet that opens
    Close Excel to see the chart in PowerPoint

    Common Chart Types

    Column Chart - Compare values across categories
    Line Chart - Show trends over time
    Pie Chart - Show parts of a whole
    Bar Chart - Compare values horizontally
    Chart Selection Tip: Choose the chart type that best represents your data. Use column charts for comparisons, line charts for trends over time, pie charts for proportions, and bar charts for ranking items.

  • 1 Applying Animations to Objects

    • Bringing Your Presentation to Life with Animations

    Animations are visual effects applied to individual elements on a slide. They can help control the flow of information and keep your audience engaged.

    Animations add movement and visual interest to slide elements

    Animation Types

    PowerPoint offers four main types of animations:

    Entrance

    How an object enters the slide

    Examples: Fade, Fly In, Zoom

    Emphasis

    Draws attention to an already visible object

    Examples: Pulse, Spin, Grow/Shrink

    Exit

    How an object leaves the slide

    Examples: Fade, Fly Out, Disappear

    Motion Paths

    Moves an object along a predefined path

    Examples: Lines, Arcs, Loops

    Applying Animations

    To add animations to objects on your slides:

    Step 1: Select Object

    Click on the text, image, or shape you want to animate

    Step 2: Choose Animation

    Go to Animations tab
    Select an animation from the gallery

    Step 3: Customize Animation

    Use Effect Options to modify direction
    Set timing (Start, Duration, Delay)
    Use Animation Pane for complex sequences

    Animation Best Practices

    Use Animations Purposefully: Animations should enhance your message, not distract from it. Use subtle animations for professional presentations and save more dramatic effects for informal or creative presentations.
    Avoid Animation Overload: Using too many different animations can make your presentation look unprofessional and distract your audience from your content. Stick to 2-3 consistent animation styles throughout your presentation.

    Animation Timing Options

    Start Option Description Use Case
    On Click Animation starts when you click the mouse When you want full control over timing
    With Previous Animation starts at the same time as previous animation For simultaneous animations
    After Previous Animation starts after previous animation finishes For sequenced animations
  • 2 Slide Transitions and Timing

    • Creating Smooth Transitions Between Slides

    Transitions are visual effects that occur when you move from one slide to the next during a presentation. They can help create a polished, professional flow.

    Transitions control how slides change during a presentation

    Applying Transitions

    To add transitions between slides:

    Step 1: Select Slides

    Select the slide(s) you want to apply transition to

    Step 2: Choose Transition

    Go to Transitions tab
    Select a transition from the gallery

    Step 3: Customize Transition

    Set Effect Options (direction, style)
    Set timing (Duration, Sound)
    Choose Advance Slide option (On Mouse Click or After)

    Transition Categories

    Subtle

    Simple, professional transitions

    Examples: Fade, Push, Wipe

    Exciting

    More noticeable transitions

    Examples: Random Bars, Vortex, Ripple

    Dynamic Content

    Transitions that move content separately from background

    Examples: Pan, Conveyor, Ferris Wheel

    Automatic Slide Advance

    You can set slides to advance automatically after a specified time:

    Setting Automatic Timing

    Select the slide(s) you want to set timing for
    In Transitions tab, uncheck "On Mouse Click"
    Check "After" and set the time (e.g., 00:05.00 for 5 seconds)
    Click "Apply To All" to use the same timing for all slides

    Rehearsing Timings

    PowerPoint can help you practice and set appropriate slide timings:

    Using Rehearse Timings

    Go to Slide Show tab → Rehearse Timings
    Practice your presentation naturally
    PowerPoint records time spent on each slide
    Save the timings when finished
    Professional Transition Tip: For business presentations, use subtle transitions like "Fade" or "Push" consistently throughout your presentation. Avoid using different transitions for each slide as this can be distracting.
    Timing Consideration: When using automatic slide advancement, make sure to allocate enough time for your audience to read and understand each slide. A good rule of thumb is 1-2 minutes per slide for text-heavy slides, and 30-60 seconds for visual slides.
