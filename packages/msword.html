Introduction to Microsoft Word

1.1 What is Microsoft Word?

Microsoft Word is a powerful word‑processing software used to create, edit, format, and share documents such as letters, reports, CVs, proposals, books, and more.

1.2 Why Learn Microsoft Word?

  • Essential for school and office work
  • Used for professional documents
  • Allows automation through templates and mail merge
  • Integrate with other Microsoft Office tools

1.3 Key Features at Glance

Text Formatting

Fonts, sizes, colors, styles, and effects

Page Design

Margins, orientation, layout, and sections

Tables & Graphics

Create and manage tables, insert images

Collaboration

Track changes, comments, and review tools

Export Options

Export to PDF, HTML, and other formats

Automation

Macros, templates, and mail merge

1.4 Real‑World Examples of Word Usage

  • Writing a CV
  • Preparing business letters
  • Creating invoices
  • Formatting academic reports
  • Designing certificates and forms

Basic Document Creation

3.1 Typing and Editing Text

Accurate text input and efficient editing are foundational skills in any word processor:

  • Place the cursor where you want to type:
    Click at the desired location in your document to position the blinking insertion point (cursor). New text will appear at this spot.
  • Use Backspace to delete left and Delete to delete right:
    • The Backspace key removes the character immediately before (to the left of) the cursor.
    • The Delete key removes the character immediately after (to the right of) the cursor.
  • Use Ctrl + Arrow Keys to move word-by-word:
    • Ctrl + ← / →: Jump the cursor one word at a time left or right.
    • Ctrl + ↑ / ↓: Move the cursor quickly between paragraphs.

3.2 Selecting Text

Selecting text is the first step for editing, copying, formatting, or deleting:

  • Click and drag to highlight:
    Position the mouse pointer at the start of the desired text, hold down the left mouse button, and drag to the end of the text to select it.
  • Double-click a word:
    Quickly select an entire word by double-clicking anywhere within it—ideal for quick edits or formatting.
  • Triple-click a paragraph:
    Triple-click anywhere inside a paragraph to select all the text in that paragraph, including the paragraph mark.

Text Formatting & Editing

4.1 Using the Font Group

The Font group (typically found on the Home tab) provides quick access to essential text formatting options:

  • Change font, size, color:
    • Font: Choose from various typefaces (e.g., Calibri, Times New Roman, Arial)
    • Font size: Adjust the size (e.g., 11 pt, 14 pt)
    • Font color: Apply a color to text via the "Font Color" button
  • Apply Bold, Italic, Underline:
    • Bold (B) makes text darker and heavier
    • Italic (I) slants text, often used for emphasis
    • Underline (U) draws a line beneath text
  • Apply Text Highlight Color:
    The highlight tool adds a colored background behind text, useful for drawing attention.

4.2 Text Effects

Text effects add artistic or decorative styling:

  • Shadow: Adds a faint, offset copy of the text behind the original
  • Glow: Surrounds text with a soft, colored halo
  • Outline: Removes the fill color from text, leaving only the edges

Paragraph & Page Layout

5.1 Paragraph Formatting

Effective paragraph formatting enhances readability and visual structure:

  • Alignment: Controls how text is positioned horizontally
    • Left alignment (default) aligns text along the left margin
    • Center alignment centers text between margins
    • Right alignment aligns text along the right margin
    • Justify alignment stretches lines so both edges align
  • Line spacing: Determines vertical space between lines
    • Single spacing (no extra space)
    • 1.5 spacing (one and a half times line height)
    • Double spacing (twice the height of line)
  • Indents and Tabs: Create visual hierarchy
    • First-line indent moves only the first line
    • Hanging indent indents all lines except the first
    • Left/right indents shift entire paragraph
    • Tabs allow precise positioning of text

5.2 Page Layout Tools

Page layout tools define the physical appearance of a document:

  • Page margins: The blank space around page edges (standard: 1 inch)
  • Orientation: Portrait (taller than wide) or Landscape (wider than tall)
  • Paper size: Dimensions (e.g., Letter 8.5" × 11", A4 210 mm × 297 mm)

Working with Styles

6.1 What Are Styles?

Preset formatting rules for headings and body text that ensure consistency and enable automatic features.

6.2 Applying and Modifying Styles

Apply Heading 1–3:

  1. Select the text you want to format
  2. Go to the Styles Gallery (Home tab)
  3. Click Heading 1 for main titles, Heading 2 for subsections, etc.

Benefits: These styles support navigation, table of contents generation, and document outlining.

Modify font, color, spacing:

  1. Right-click a style in the Styles Gallery
  2. Choose Modify
  3. Adjust font type, size, color, spacing, alignment
  4. Check "Automatically update" if desired
  5. Click OK to save

6.3 Creating New Styles

Use Styles Pane → New Style to define custom styles:

  1. Format a sample paragraph exactly as desired
  2. Select the formatted text
  3. Open the Styles Pane (click arrow in Styles group)
  4. Click New Style (+ icon)
  5. In the dialog box, name your style and adjust formatting
  6. Choose whether it applies to paragraphs or characters
  7. Click OK

Tables Creation & Management

7.1 Creating a Table

Tables organize and present data clearly in rows and columns:

  • Use Insert → Table:
    Click the Insert tab, select Table, then choose rows and columns

7.2 Editing Tables

Refine table structure and appearance:

  • Insert rows/columns: Right-click inside a cell and choose Insert options
  • Merge cells: Select adjacent cells and click Merge Cells
  • Apply table styles: Use Table Design tab for professional formatting

7.3 Sorting Data

Organize information logically:

  1. Click anywhere inside the table
  2. Go to Layout → Sort
  3. Choose to sort by columns (A-Z, Z-A, numbers)
  4. Ensure "Header row" is checked if first row has titles

Graphics & Media Integration

8.1 Inserting Images

Images enhance visual appeal and convey information effectively:

  • From device: Insert → Pictures → This Device
  • Online images: Insert → Online Pictures (Bing search)
  • Stock images: Built-in access to royalty-free photo libraries

8.2 Formatting Images

Style and position images to integrate with text:

  • Wrap text: Controls how text flows around image
    • Square: Text wraps in rectangular shape
    • Tight: Text follows image outline
    • Behind Text / In Front of Text: For watermark or overlay
  • Crop: Trim unwanted parts (Picture Format → Crop)
  • Add borders/effects: Enhance with outlines, shadows, reflections, filters

8.3 SmartArt and Shapes

Insert vector-based graphics to illustrate ideas:

  • SmartArt: Pre-designed diagrams for processes, hierarchy, cycles
  • Shapes: Draw lines, arrows, callouts, rectangles (Insert → Shapes)

Document References & Organization

9.1 Table of Contents (TOC)

A Table of Contents provides a navigational roadmap for long documents:

  • Uses heading styles: Automatically generated from Heading 1, 2, 3
  • Only text with heading styles appears in TOC
  • Modifications automatically update when refreshed

To insert a TOC:

  1. Place cursor where you want TOC (usually beginning)
  2. Go to References → Table of Contents
  3. Choose built-in style or customize

Tip: Always update TOC (right-click → Update Field) after changes.

9.2 Footnotes & Endnotes

Provide supplementary information without interrupting main text:

  • Footnotes: Appear at bottom of same page
  • Endnotes: Appear at end of document or section

To add:

Go to References → Insert Footnote or Insert Endnote

9.3 Citations & Bibliography

Essential for research-based writing:

  • Insert citations: References → Insert Citation → Add New Source
  • Formats according to chosen style (APA, MLA, Chicago)
  • Generate bibliography: References → Bibliography
  • Creates complete, alphabetized reference list

Mail Merge Mastery

10.1 What Is Mail Merge?

A powerful feature that automates personalized documents by combining a main document with a data source.

Common uses:

  • Personalized form letters
  • Mailing labels and envelopes
  • Email messages

10.2 Connecting to a Data Source

Supported data sources:

  • Excel: Most common (store names, addresses in rows/columns)
  • Outlook Contacts: Use email address book
  • Access: For advanced relational databases

To connect:

  1. Start Mail Merge from Mailings tab
  2. Choose document type (e.g., Letters)
  3. Click Select Recipients → Use an Existing List
  4. Browse to Excel file and choose worksheet

Tip: Ensure Excel sheet has clear headers and no blank rows.

10.3 Preview and Finish

  • Preview Results: Mailings → Preview Results to see personalized versions
  • Finish & Merge: Generate new document with all letters, print, or send emails

Exercise 10.1 – Generate a Letter Merge

Task: Create personalized form letters for 5 people using Excel data.

Steps:

  1. Prepare Excel sheet with columns: FirstName, LastName, City
  2. Open new Word document → Go to Mailings tab
  3. Click Start Mail Merge → Letters
  4. Click Select Recipients → Use an Existing List
  5. Type letter with merge fields: Dear «FirstName», ... cities like «City»!
  6. Click Preview Results to verify
  7. Click Finish & Merge → Edit Individual Documents

Result: New document with 5 personalized letters ready to print or email.

Advanced Features & Automation

11.1 Quick Parts

Reusable pieces of content (text, graphics, tables) for quick insertion:

  • Examples: Company address, email signatures, standard paragraphs
  • To save: Highlight content → Insert → Quick Parts → Save Selection
  • To insert: Insert → Quick Parts → Choose saved items

Tip: Useful for maintaining consistency in templates.

11.2 Macros

Recorded sequences of actions that can be replayed with one command:

  • Ideal for automating: Custom formatting, table insertion, layout changes
  • Written in VBA (Visual Basic for Applications) but can be recorded without coding
  • Security Note: Only enable macros from trusted sources

11.3 Form Controls

Turn static documents into interactive templates:

  • Common controls (Developer tab):
    • Check Box Content Control
    • Drop-Down List Content Control
    • Text Box (Rich Text or Plain Text)
  • Protect form: Review → Restrict Editing to prevent editing non-form areas

Note: Enable Developer tab in ribbon to access these tools.

Collaboration & Review Tools

12.1 Track Changes

Collaborative editing feature that visually records all modifications:

  • When enabled (Review → Track Changes), edits appear in color
  • Insertions: Underlined or colored text
  • Deletions: Strikethrough or deleted text balloon
  • Each reviewer's changes shown in different color
  • Accept or reject individual or all changes
  • Ideal for team editing and peer review

12.2 Comments

Add non-intrusive notes without altering original text:

  • To add: Select text → Review → New Comment
  • Type feedback in comment balloon in margin
  • Comments linked to user name, can be replied to or resolved
  • Use for suggesting rewrites, asking questions, flagging issues

Tip: Comments don't appear in printed documents unless specified.

12.3 Compare Documents

Automatically detect differences between two document versions:

  • Go to Review → Compare → Compare...
  • Select Original and Revised documents
  • Generates third document showing all changes
  • Includes summary of differences
  • Invaluable for version control in legal, academic work
