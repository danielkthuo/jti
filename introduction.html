Introduction to Computers

A computer is a multipurpose electronic device that can receive, process and store data.

It operates under the control of instructions stored in its own memory, that can accept data, manipulate the data according to specified rules, produce results, and store the results for future use.

It has input, processing, storage and output. Examples of computers could be:

  • Desktop computers
  • Laptops
  • Smart phones
  • Office calculator
  • Point of Sale systems

Powering a Computer

In order for a computer to function, it must be powered. The two ways of powering are:

  • Alternating Current (AC): Supplied as electricity
  • Direct Current (DC): Supplied as a battery

Some devices can use both, others have an option of either, based on power consumptions.

Booting Process

Powering a computing involves booting, which can further be classified as:

  • Cold booting: When the computer is started after having been switched off
  • Warm booting: When the operating system alone is restarted (without being switched off)

Computer Devices

Input, Output and Input/Output Devices

Device Type Description Examples
Input Device A device that gives instructions or data to a computer for processing. Usually controlled by user to give instructions to a computer. Can send data to another device, but cannot receive data from another device. Keyboards, Mouse, Scanner, Camera, etc.
Output Device A peripheral component that the processor uses to display information to the user or another device. Can receive data from another device, but cannot send data to another device. Monitor (Visual Display Unit), Projector, Speakers, etc.
Input/Output Device A device that allows entering of data and also displaying or outputting data/information. Can send data to another device and also receive data from another device. Headsets (with speakers and microphone), Facsimile (fax), Touch screen monitor, etc.

Computer Software

Software (SW/S/W) is a collection of instructions that enable the user to interact with a computer, its hardware, or perform tasks.

Operating System (OS)

An operating system, or "OS," is software that communicates with the hardware and allows other programs to run. It is comprised of system software, or the fundamental files your computer needs to boot up and function.

Every desktop computer, tablet, and smartphone includes an operating system that provides basic functionality for the device.

Examples of Operating Systems:

  • Mac OS X
  • Windows XP
  • Windows 7
  • Windows 10
  • Linux
  • Android
  • iOS

Application Software

Application software, also known as an "application" or an "app", is computer software designed to help the user to perform specific tasks. This is the type of software that interfaces with the user. It is the mode of the communication with the user.

Types of Computer Software:

  • Internet browser - Firefox, Google Chrome, Internet Explorer
  • Programming language - C++, HTML, Java, Perl, Visual Basic (VB)
  • Word processors (Microsoft Word, Google Docs)
  • Spreadsheet software (Microsoft Excel)
  • Presentation software (Microsoft PowerPoint)
  • Graphics software (Adobe Photoshop)

Functions of Operating Systems

  • Manage hardware resources - these include managing memory (RAM and ROM), printers, screen, keyboard mouse etc.
  • Providing user interface - the OS enables the users to see where files are located, and allows the user to work with the computer hardware through the graphical user interface.
  • Provide interface between application software and machine - when a file is created by an application e.g. MS Word, when the user selects save, it is the operating systems that writes the file on the disk.
  • Provide security - the operating system provides user authentication, and ensures malicious programs do not get into the system.
  • Provide utility software for maintenance - the operating system monitors how other programs are installed and running. If something goes wrong, the OS helps to rectify.

File Compression

Zipped (compressed) files take up less storage space and can be transferred to other computers more quickly than uncompressed files.

Why do we compress files?

  1. It allows better back-up of huge files because of the reduced capacity of the files
  2. Allows easier and cheaper file transfers because of using less bandwidth
  3. It allows "lighter" files to run on the web and therefore increases the speed of website loading and unloading
  4. Compression allows data encryption and therefore provided enhanced file security

Computer Networks

A computer network is a group of interconnected computers and devices that communicate and share resources such as files, printers, internet, and applications.

Types of Computer Networks:

  • LAN (Local Area Network): Covers a small area like an office or building
  • WAN (Wide Area Network): Spans large geographical areas (e.g., the internet)
  • MAN (Metropolitan Area Network): Covers a city or campus
  • PAN (Personal Area Network): Very small, like Bluetooth connections between phone and headset

Key Network Components:

  • Router: Directs data between networks
  • Switch: Connects devices within the same network
  • Modem: Connects the network to the internet
  • Access Point: Extends wireless connectivity

Computer Security

Computer security (cyber security / IT security) is the protection of computer systems from the theft or damage to the hardware, software or the information on them, as well as from disruption or misdirection of the services they provide.

Key Areas of Computer Security:

  • Data Security: Protecting data from unauthorized access or corruption (e.g., encryption, backup)
  • Network Security: Preventing unauthorized access and threats on a computer network (e.g., firewalls, intrusion detection systems)
  • Application Security: Ensuring software applications are secure from vulnerabilities (e.g., regular updates, secure coding practices)
  • User Security: Educating and controlling how users access systems (e.g., strong passwords, multi-factor authentication)
  • Device Security: Securing physical and digital access to computers and mobile devices (e.g., antivirus, screen locks)

NIST Cybersecurity Framework

In order to provide sufficient security, an organization needs to follow these steps based on National Institute of Standards and Technology (NIST) Cybersecurity Framework:

  1. IDENTIFY - Take inventory of key technologies you use and know what information you need to rebuild your infrastructure from scratch
  2. PROTECT - Assess what protective measures you need to have in place to be as prepared as possible for a cyber incident
  3. DETECT - Put measures in place to alert you about current or imminent threats to system integrity, or loss or compromise of data
  4. RESPOND - Make and practice an incidence response plan to contain an attack or incident and maintain business operations in the short term
  5. RECOVER - Know what to do to return to normal business operations after an incident. Protect sensitive data and your business reputation over the long term

Common Threats:

  • Viruses and Malware: Malicious programs that can damage or steal data
  • Phishing Attacks: Fraudulent attempts to gain sensitive information
  • Ransomware: Malicious software that locks files until a ransom is paid
  • Hackers: Unauthorized users who try to gain access to systems

Security Measures:

  • User authentication - ensure that only authorized users access the systems. This can be achieved with usernames and password
  • Firewalls - this ensures that devises and users outside the network are allowed entry after checking their safety
  • Antivirus - checks that viruses do not enter the network
  • Email scans - ensures that junk mail and malicious emails filtered before getting into the network
  • Keep your software updated - Ensure you download the latest operating system patches and keep all software up-to-date

Hardware Protection Measures:

  • Fire - have preventive strategies against fire. Ensure you have different types of fire extinguishers for different types of fire and use them appropriately
  • Extreme temperatures - Most computer systems should be kept between 50 and 90 degrees Fahrenheit (10 to 32 degrees Celsius). Have free flowing air to maintain this temperature or install air-conditioning units
  • Electrostatic Discharge (ESD) - damages electronics like memory chips, motherboard, processors etc. Store this hardware in static shielding bags. Use static traps or ground bracelets
  • Smoke - Smoke is a potent abrasive and collects on the heads of magnetic disks, optical disks, and tape drives. Ensure you have installed smoke detectors

Computer Power Management

Computer power management is a set of features that help reduce power consumption by placing monitors and computers into a low-power "sleep mode" after a period of activity. Simply touching the mouse or keyboard "wakes" the computer and monitor in seconds.

Types of Power Management

There are 4 basic types of computer power management or "sleep" features on Windows PCs:

Type Description Benefits
System standby Drops monitor and computer power use down to 1–3 watts each. Wakes up in seconds Saves $25–75 per PC annually
System hibernates Drops monitor and computer power use down to 1–3 watts each. Wakes up in 20+ seconds. Saves work in the event of power loss Saves $25–75 per PC annually
Turn off monitor Drops monitor power use down to 1–3 W. Wakes in seconds or less Saves half as much as system standby or hibernate: about $10–40
Turn off hard disks Saves very little energy Minimal savings

Benefits of Power Management

  1. Cutting the electricity used by PCs roughly in half, saving $25–75 per PC annually
  2. Enhancing data security by reducing the chance that valuable personal information is displayed on an unattended PC
  3. Saving time by eliminating the daily wait for computers to boot up
