Text Formatting - The "What"

Controls how individual characters and words appear.

Font Type
Arial
Times New Roman
Calibri

Stick to 1-2 professional fonts per document

Font Size
12pt - Body text
14pt - Subheadings
18pt - Main headings

Use larger sizes for headings to create hierarchy

Font Styles
Bold for emphasis
Italic for titles or foreign words
Underline for hyperlinks only

Avoid excessive styling - less is more!

Colors & Highlights
Font Color for emphasis
Highlight for key points

Use sparingly and maintain readability

Styles - The Professional's Secret Weapon

What are styles? Pre-defined formatting combinations that can be applied with one click.

Without Styles
  • Manually format each heading
  • Inconsistent sizes and spacing
  • Hard to update globally
  • No automatic table of contents
With Styles
  • One-click consistent formatting
  • Easy global updates
  • Automatic document structure
  • Professional appearance
Heading 1 - Main document titles (24pt, bold)
Heading 2 - Major section headings (18pt, bold)
Heading 3 - Subsection headings (14pt, bold)
Normal - Body text (11-12pt, regular)
Teaching Tip: Use the house construction analogy - styles are like using pre-built walls and windows instead of cutting every piece of wood individually.

Common Challenges & Solutions

Issue: Text moves unpredictably when inserting images

Solution: Use proper text wrapping instead of "In Line" positioning

Issue: Can't find the Styles menu

Solution:
Google Docs: Look between Zoom and Paint Format tools
Microsoft Word: Home tab ‚Üí Styles gallery

Issue: Comments not showing up for collaborators

Solution: Check sharing permissions - need at least "Commenter" access

Issue: Formatting looks different on other computers

Solution: Use standard fonts (Arial, Times New Roman) and export as PDF for consistent appearance

Advanced Tips for Quick Learners

Format Painter: Copy formatting from one selection and apply to another

Keyboard Shortcuts: Ctrl+Shift+C (Copy format), Ctrl+Shift+V (Paste format)

Find and Replace: Ctrl+H to quickly change repeated words or phrases

Columns: Create newspaper-style layouts for newsletters and brochures

Templates: Start with pre-designed templates for resumes, letters, and reports
