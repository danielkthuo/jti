Core Concepts
What is a Computer?

A computer is an electronic machine that:

  • Accepts data (input) from users or other sources
  • Processes that data according to instructions
  • Produces new information (output) as a result
Teaching Tip: Use the human body analogy - brain (processing), senses (input), voice/actions (output).

The Operating System (OS)

Analogy: The Manager or Conductor

The OS is software that manages all hardware resources and provides a platform for applications to run.

Key Point: Emphasize that without an OS, the computer is just expensive hardware that can't do anything useful.

Common Operating Systems:

  • Windows (Microsoft) - Most common on PCs
  • macOS (Apple) - On Mac computers
  • Chrome OS (Google) - On Chromebooks
  • Linux - Various distributions

Common Questions & Troubleshooting

Student Question: "Why does my computer get slow when I have many programs open?"

Answer: Explain RAM limitations - when RAM fills up, the computer uses slower storage as "virtual memory."

Student Question: "What's the difference between shutting down and restarting?"

Answer: Shutting down turns the computer completely off. Restarting shuts down and turns back on, which can fix many software issues.

Issue: Student can't find File Explorer/Finder

Solution:
Windows: Press Windows key + E
macOS: Press Command + Space, type "Finder"

Core Concepts
Files vs. Folders
üìÑ
File

A single item containing data

  • Document (.docx, .pdf)
  • Image (.jpg, .png)
  • Audio (.mp3, .wav)
  • Video (.mp4, .mov)

Has a name and extension that tells the computer what type of file it is

üìÅ
Folder

A container for organizing files

  • Can contain files
  • Can contain other folders (subfolders)
  • Helps keep things organized
  • Has a name but no extension
Analogy: Files are like individual papers, folders are like filing cabinet drawers or folders that hold those papers.
Essential File Operations
Create: Right-click ‚Üí New ‚Üí Folder/Text Document
Rename: Select file ‚Üí F2 or Right-click ‚Üí Rename
Copy: Ctrl+C (makes duplicate) ‚Üí Navigate to destination ‚Üí Ctrl+V
Move: Ctrl+X (cuts original) ‚Üí Navigate to destination ‚Üí Ctrl+V
Delete: Select file ‚Üí Delete key (goes to Recycle Bin/Trash)
Permanent Delete: Shift+Delete (bypasses Bin/Trash - use with caution!)
Keyboard Shortcuts to Teach:
Ctrl+C (Copy), Ctrl+X (Cut), Ctrl+V (Paste), Ctrl+Z (Undo)
F2 (Rename), Delete (Move to Recycle Bin)
Safe Software Installation

Official Sources Only:

  • Microsoft Store (Windows)
  • Apple App Store (macOS)
  • Official developer websites

Risks of Unofficial Sources:

  • Viruses and malware
  • Spyware that steals information
  • Ransomware that locks your files
  • Poorly working or fake software
Teaching Tip: Use the analogy of buying food - you wouldn't buy from a random street vendor with no credentials, you'd go to a reputable store.

Common Challenges & Solutions

Issue: Student can't see file extensions

Solution:
Windows: File Explorer ‚Üí View ‚Üí Check "File name extensions"
macOS: Finder ‚Üí Preferences ‚Üí Advanced ‚Üí Check "Show all filename extensions"

Issue: Student accidentally deleted important file

Solution: Check Recycle Bin (Windows) or Trash (macOS) and restore

Issue: Student can't find a file they saved

Solution:
Use search function in File Explorer/Finder
Check default save locations (Documents, Downloads, Desktop)

Assessment & Reinforcement

Formative Assessment Questions:

  • "What's the difference between copying and moving a file?"
  • "Why should we use folders to organize our files?"
  • "Where should you download software from and why?"
  • "What happens when you press Delete vs. Shift+Delete?"

Reinforcement Activity: "File Organization Challenge" - Time students to organize a new set of chaotic files using the skills they learned.

Module 1 Summary & Transition

Key Takeaways:

  • Computers process input to create output using hardware components
  • The OS manages all hardware and software interactions
  • Proper file organization saves time and reduces frustration
  • Safe computing practices protect your system and data
