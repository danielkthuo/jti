Data Entry & Navigation
Entering Data: Click cell and type, then press Enter (moves down) or Tab (moves right)
Editing Data: Double-click cell or press F2
Selecting Multiple Cells: Click and drag, or Shift + Arrow keys
Selecting Entire Sheet: Click corner button between A and 1, or Ctrl+A
Navigation Shortcut Action
Enter Move down one cell
Tab Move right one cell
Shift+Enter Move up one cell
Shift+Tab Move left one cell
Ctrl+Arrow Jump to edge of data region
Ctrl+Home Go to cell A1

Basic Formulas - The Magic Begins!

Golden Rule: ALL formulas start with an equal sign =

SUM

Adds up a range of numbers

=SUM(B2:B10)

Adds all numbers from cell B2 through B10

Use for: Totals, running sums

AVERAGE

Calculates the mean of numbers

=AVERAGE(B2:B10)

Finds average of numbers from B2 through B10

Use for: Grades, performance metrics

COUNT

Counts how many cells contain numbers

=COUNT(B2:B10)

Counts cells with numbers from B2 through B10

Use for: Attendance, inventory counts

MAX/MIN

Finds highest/lowest value

=MAX(B2:B10)
=MIN(B2:B10)

Use for: Finding best/worst performance

The Fill Handle - Superpower Tool
‚¨áÔ∏è
Fill Handle

The small square at the bottom-right corner of a selected cell

Drag down to copy formulas to multiple cells

Smart filling - automatically adjusts cell references

Type formula in first cell: =B2*C2
Click cell to select it
Hover over bottom-right corner until cursor changes to +
Click and drag down to fill other cells
Watch formulas automatically adjust: =B3*C3, =B4*C4, etc.
Teaching Tip: The fill handle is one of the most powerful time-saving features. Emphasize how it eliminates repetitive typing and prevents errors.

Common Challenges & Solutions

Issue: Formula shows error (#VALUE!, #DIV/0!)

Solution: Check for text in number cells, empty cells in calculations, division by zero

Issue: Fill handle not working properly

Solution: Ensure you're dragging the small square in corner, not the cell border

Issue: Chart shows wrong data

Solution: Reselect correct data range, check if headers are included

Issue: Sorting mixes up data

Solution: Always select entire data range including headers before sorting

Issue: Filter not showing all options

Solution: Clear existing filters, ensure no blank rows in data

Advanced Tips for Quick Learners

Conditional Formatting: Automatically color cells based on values (great for highlighting high/low numbers)

Sparklines: Mini charts within single cells to show trends

Data Validation: Restrict what can be entered in cells (dropdown lists, number ranges)

Pivot Tables: Advanced data summarization (introduce concept only)

Named Ranges: Give cell ranges meaningful names for easier formula writing
