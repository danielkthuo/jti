Introduction to Word Processors

According to Webopedia, word processing is the phrase used to describe using a computer to create, edit, and print documents. Of all computer applications, word processing is the most common. To perform word processing, you need a computer, a special program called a word processor, and a printer.

Popular Word Processors

  • Microsoft Word - Part of the Microsoft Office suite
  • Google Docs - Free online word processor from Google
  • LibreOffice Writer - Free and open-source alternative
  • Apple Pages - For macOS and iOS users

Key Features of Word Processors

  • Text creation and editing
  • Formatting options (fonts, sizes, colors)
  • Paragraph alignment and spacing
  • Spell checking and grammar correction
  • Insertion of images, tables, and other objects
  • Page layout and design options
  • Templates for quick document creation
  • Collaboration features

Interface Components Explained

  • Title Bar - Shows the document name and application name
  • Menu Bar - Contains menus for various functions
  • Toolbars - Provide quick access to frequently used commands
  • Text Area - The main workspace where you type and edit text
  • Status Bar - Shows information about the current document

Window Controls

  • Minimize Button (-) - Makes the window disappear without closing it
  • Restore Down (□) - Changes the window size without closing it
  • Close (X) - Closes the window. If the document has been modified, it prompts to save changes

Tool Tips

A tool tip is a message that pops up when the mouse pointer is passed over a tool. For example, hovering over the Bold button shows "Bold (Ctrl+B)".

Online Text Editors

Google Docs

To use an online text editor, you need an active Google account. Google accounts are created for free, but if a user wants to use enhanced features, they can pay for those features. Google's "Docs" is a free online text editor which you launch from the Google Apps menu after logging into your account.

Click on Docs → Start a new document. A new "Untitled document" is created with a similar but slightly different menu and toolbars from Microsoft Word text editor.

Advantages of Online Text Editors

  • Accessible from any device with an internet connection
  • Automatic saving and version history
  • Real-time collaboration features
  • No installation required
  • Free basic versions

Document Views and Window Splits

MS Word allows users to view documents in five different formats, each with its own features/advantages.

Document Views in Microsoft Word

  • Print Layout – Allows the user to view the document as it will look when printed. It shows the margins, headers and footers and any formatting.
  • Full Screen Reading – This view enables the user to get a feel of reading a book. It eliminates the menu and tool bars for ease of reading and uses only the tools that help navigate the document.
  • Web Layout – Displays the document text as it would appear on a website, should the user publish the document as is. It imitates the display of a web browser. The view will vary slightly based on the browser one is using.
  • Outline – This view enables the user to see the "Levels" that the document has. The extent to which the text is indented from the margins is shown and any formatting that has title headers. Page breaks and section breaks are also visible but images are not visible. It is used to format the section and page breaks.
  • Draft – This view is used to format a document quickly without concern on how it will actually look in print form. This view eliminates headers and footers, pictures that do not have text wrap style.

Zooming In and Out

The zoom feature allows for magnification of an application for better view. You can zoom in to get a close-up view of your file or zoom out to see more of the page at a reduced size. You can also save a particular zoom setting with a document or template, presentation, or worksheet.

To zoom in or zoom out, on the status bar, click the Zoom slider. Hold the left and drag the mouse towards left (-) to zoom out and towards right (+) to zoom in. On the View menu, you can interact with the zoom more.

Split Window Feature

By splitting up the document view, you can navigate to different parts of the document in different views. If you need to see different parts of a document, you can use the split feature to separate the document.

  4. You can adjust the location of the split. Hold down the left mouse button and move the split bar

Document Security

Securing a document is a feature found in word processors as part of Electronic Data Management (EDM). The same feature is available for other applications in the suite (e.g., PowerPoint, Access, Excel etc.) and the principle of applying security is the same. When a user does not want their document to be opened and/or edited without authorization, word processor enables the user to require authentication.

Class Activity: Applying Document Security

  1. Open the document "CAT Paper.Docx" in the class folder. Immediately, the content of the examination are revealed. This means that if the document was accessed by anybody, the confidentiality of the document is lost.
  2. Save the file as "CAT Paper Secured.Docx"
  3. To secure the document, select File → Info → Permissions → Protect Document → Encrypt with Password
  4. Type in your password that you would like to use. Then confirm the password and select OK.
  5. Close the dialogue box window, when prompted to save changes, select Yes.
  6. Close the document CAT Paper Secured.Docx.
  7. Open the document CAT Paper Secured.Docx. It will prompt you to specify the password.

Note: If you lose or forget a password, Word cannot recover your data.

Restricting Editing

  1. Open the document CAT Paper Secured.Docx and enhance the document security
  2. Select File → Info → Permissions → Protect Document → Restrict Editing
  3. Select by checking the Check Box, No Changes (Read Only)
  4. In Start Enforcement select Yes, Start Enforcing Protection
  5. You will be prompted for password. Enter new password different from encryption password. (Same password will still work)
  6. Select OK, then close the documents and Save Changes
  7. Reopen the document and attempt to make changes. MS Word will restrict typing. On the right, there is a dialogue menu that allows you to stop protection

Working with Tables

In Microsoft Word, you can add a table in a document in three ways:

  1. The Insert → Table → Insert Table menu allows a user to specify the number of rows and columns (table dimension)
  2. The Insert → Table → Draw menu allows a user to use a "pen" to draw the table rows and columns. This menu is also used to split rows and columns in an existing table.
  3. The Insert → Table → Quick Tables menu allows a selection of preformatted tables. The tables have sample data that allows user to have a "feel" of the table type and format they would like. Users choose from a gallery of preformatted table templates.

Using Thesaurus

In general usage, a thesaurus is a reference work that lists words grouped together according to similarity of meaning (containing synonyms and sometimes antonyms), in contrast to a dictionary, which provides definitions for words, and generally lists them in alphabetical order.

Using the thesaurus, you can look up synonyms (different words with the same meaning) and antonyms (words with the opposite meaning).

Using Word Count Tool

When you type a document, Word automatically counts the number of pages and words in your document and displays them on the status bar at the bottom of the workspace. If you don't see the word count in the status bar, right-click the status bar, then and click Word Count.

Microsoft Word counts the number of words in a document while you type. Word can also count the following:

  • Pages
  • Paragraphs
  • Lines
  • Characters, either including or excluding the spaces

When the document is active, there is an automatic word count at the left bottom side of the Word application.

  1. In the Review tab, Proofing category, select Word Count
  2. A dialogue box window appears with all the statistics of the documents
